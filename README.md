# 🍔 Sistema de Hamburguesas en Java (POO)

<p align="center">
Aplicación básica desarrollada en <b>Java</b> para practicar <b>Programación Orientada a Objetos (POO)</b> mediante la creación y gestión de hamburguesas con diferentes ingredientes.
</p>

---

## 📖 Descripción del Proyecto

Este proyecto consiste en un pequeño sistema desarrollado en **Java** que permite crear objetos de tipo **Hamburguesa**, definir sus ingredientes y mostrar su información en la consola.

El objetivo principal es practicar los fundamentos de la **Programación Orientada a Objetos (POO)** aplicando conceptos como:

- Clases
- Objetos
- Atributos
- Constructores
- Encapsulamiento
- Métodos
- Getters y Setters
- Uso de colecciones (`ArrayList`)

Cada hamburguesa puede contener distintos ingredientes como pan, tipo de carne, vegetales, queso, tocineta y salsas, además de un precio asociado.

---

## 🧠 Conceptos de Programación Aplicados

| Concepto | Explicación |
|--------|-------------|
| **Clase** | Modelo que define las características de una hamburguesa |
| **Objeto** | Instancia creada a partir de la clase |
| **Atributos** | Variables que describen las características de la hamburguesa |
| **Encapsulamiento** | Uso de atributos `private` para proteger los datos |
| **Constructores** | Permiten crear objetos con o sin valores iniciales |
| **Métodos** | Funciones que realizan acciones dentro de la clase |
| **Getters / Setters** | Métodos para acceder y modificar los atributos |
| **ArrayList** | Estructura utilizada para almacenar múltiples hamburguesas |

---

## 📂 Estructura del Proyecto

com.sena  
│  
├── Hamburguesa.java  
│  
└── App.java  

**Hamburguesa.java**  
Contiene la clase que define las características de una hamburguesa, incluyendo atributos, constructores, getters, setters y métodos.

**App.java**  
Contiene el método `main`, donde se crean las hamburguesas, se almacenan en una lista y se muestran en la consola.

---

## 🍔 Ingredientes de las Hamburguesas

Cada hamburguesa puede incluir diferentes ingredientes:

- 🥖 Pan (artesanal, brioche, integral)
- 🥩 Tipo de carne (res, pollo, doble carne)
- 🥬 Lechuga
- 🍅 Tomate
- 🧅 Cebolla
- 🧀 Queso
- 🥓 Tocineta
- 🥫 Salsas especiales
- 💲 Precio

Estos ingredientes se almacenan como **atributos dentro de la clase `Hamburguesa`**.
